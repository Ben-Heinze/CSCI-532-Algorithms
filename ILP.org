* Integer Linear Programming

Vertex cover: given graph $G = (V,E)$ find the smallest V' $\subset$ V such that each edge in E contains an end point in V':
- $x_i \in \{0,1\}$ = indicates if vertex $i$ is selected.
- Objective: min $\sum_i x_i$
- subject to:
  - $x_i + x_j \leq 1$, for each edge $e = (i,j)$

Solving integer linear programs (ILPs) is NP-Hard (in general).

ILP vs LP:
Since the PL has more options to reduce the objective value, $OPT_{LP} \leq OPT_{ILP}$ (for a minimization problem). If the minimum objective value comes from an integer solution, a plain LP solver (e.g., Simplex) will find it.

** Solving ILPs

Supposed we have an ILP that is a maximization problem.

1. Relax ILP (turn integers into real values) $x \in \mathbb{Z} \rightarrow x \in \mathbb{R}$

2. Solve relaxed LP. Suppose the optimal objective is 100 with a fractional solution (ig some $x_i = 3.24$)
   Since the relaxed LP has more optinons to increase the objective value, $OPT_{LP} \leq OPT_{ILP}$

3. Formulate two neew LPs. One with $x_i \geq 4$ and the other one with $x_i \leq 3$ constraints. New you have two LP subproblems that you can solve easily.

4. Solve subproblem LPs

5. Formulate two new LPs that split the subproblem on some non-integer variable

6. Solve the subproblem LPs.

7. Repeat.

Main problem (LP relaxation): $x \in \mathbb{R}, obj = 100$
   
Subproblem 1 result: $x \in \mathbb{Z}, obj = 85$

Subproblem 2 result: $x \in \mathbb{Z}, obj = 90$

If both subproblems are integers ($\in \mathbb{Z}$), just select the highest one (subproblem 2 in this case).

* Vertex Cover


Total Unimodularity: A matrix is **totally unimodular** if the determinant of any square submatrix of it is 0, 1, or -1.

- This impies that totally unimodular matrices are composed of 0s 1s or -1s, since single elements are square submatrices.
- This also means that if $A$ is totally unimodular, $A^T$ is too since $det(B) = det(B^T)$


**Theorem (Hoffman-Krusal, 1956):**
- $A \in \mathbb{R}^{m \times n} \text{is totally unimodular} \iff \text{the feasible region:} \{x \in \mathbb{R}^n \mid Ax \leq b, x \geq 0 \} \text{has integer vertices } \forall b \in \mathbb{Z}^m$.

  - If we can prove that our coefficient constraint matrix is unimodular, we can solve it in polynomial time.

Objective: $max(c^T x)$
Subject to: $A x \leq b ; x \geq 0$

If A is totally unimodular, the ILP can be solved in polynomial time! However, there is no quick and easy way to determine if a matrix is unimodular until Ghoula-Houri's theorem came out:

**Theorem (Ghoula-Houri, 1962):**

- $A \in \mathbb{R}^{m \times n}$ is unimodular $\iff$ for every subset of rows R, there is a partition $R = R_1 \bigcup R_2$ such that for every column $j$, $\sum_{i \in R_1} A_{ij} - \sum_{i \in R_2} A_{ij} \in \{-1, 0, 1\}$


These two theorems are linked together by the unimodularity. We can show if a problem has the Ghoula-houri property, then it has an integer solution
